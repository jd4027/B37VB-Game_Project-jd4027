Dodge the blocks game:

Dodge the blocks is a 2D dodging game that is made using C and raylib.
In the game the player moves from left to right and tries to avoid the blocks which are falling from the top of the screen.
As the player plays the game for a longer period of time the game gets more difficult.

 How to compile:

 To compile raylib must be installed on the device.
 If you are on Windows using MINGW, you can compile the game using:
 gcc main.c -o game.exe -lraylib -lopengl32 -lgdi32 -lwinmm


 How to run:

 On windows:

 game.exe

 Controls: 

 Move Right = D
 Move Left = A
 Start/Restart Game = Space

 Requirements To Run:

 Must need a C Compiler
 Must have Raylib installed
 Device must be able to run 2D graphics.
